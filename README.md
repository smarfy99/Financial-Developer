# Financial-Developer

## 금융공부하는 개발자

----

- 기간 : 7/9 ~ 2024년까지
- 기능
  1. 블로그 글 올리기 (금융(분석, 자격증 등) + 개발 + etc)
  2. 캘린더에 그날 한 일 + 굵직한 일정 기록
  3. 자기소개
  4. (+ 자산 포트폴리오 만들기 / ex. 비과세 한도 계산기, 금리 변동에 따른 etf 추천 등)

- 기술스택
  1. 프론트엔드 : NEXT.js
  2. 백엔드 : SpringBoot (No JPA)
  3. DB : 차차 고민 좀 해봄
  4. 배포 : AWS
